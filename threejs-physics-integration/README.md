# Experimental Three.js Physics Notes

This directory contains repository-only research. It has no `SKILL.md`, is not
an installable skill, and is not an invocation or routing target. The retained
notes describe reusable mechanisms rather than a serialized cross-system
contract.

## Retained scope

- For render, query, collider, support, and integration representation
  boundaries—and for choosing a solver family—read
  [solver selection and architecture](references/solver-selection-and-architecture.md).
- For moving-frame kinematics, SE(3) force/wrench transport, and external-solver
  ownership and lifecycle, read
  [external solver boundaries](references/external-solver-boundaries.md).
